8.2.2 For Rubyists

Templating in Artanis looks very similar to Rails.

The Rails code: 

     <% if( @fullscreen == 1 ) %>
     <%= "<div class='full'><p>...</p></div>" %>
     <% end %>

And the same function in Artanis code: 

     <% (if (= fullscreen 1) %>
     <% "<div class='full'><p>...</p></div>" %>
     <% ) %>
