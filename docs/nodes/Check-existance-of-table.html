9.3.7 Check existance of table

     ;; case sensitive
     (mtable 'exists? table-name . columns)
     ;; or for case-insensitive
     (mtable 'ci-exists? table-name . columns)

For example: 

     (mtable 'exists? 'Persons 'city 'lastname)
